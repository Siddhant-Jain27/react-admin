# React Admin Example

Example of React Admin. We use JSON-Server for our REST API

## Usage

```
# Install dependencies
npm install

# Install client dependencies
cd client
npm install
cd ..

# Run client and server (Visit http://localhost:3000)
npm run dev

# Run server only (:5000)
npm run server

# Run client only (:3000)
npm run client
```
